Download มาแล้วแตกไฟลที่ไหนก็ได้แล้วก็ทำการเปิดแอปขึ้นมา
หลังจากนั้นให้พิม npm install เพื่อติดตั้ง dependencies ที่จำเป็น
ทำให้แน่ใจว่าเราเชื่อม Mongodb อยู่ตามใน readme_installation
พิม node seed.js แต่ว่าตัวฐานข้อมูลอาจพังนิดนังเพราะผมแก้ไม่เป็น

----------------------------------------------------------
