## Slide 2: ภาพรวมระบบ (System Overview)

| โมดูล | คำอธิบาย |
|---|---|
| 🔐 ระบบ Login/Logout | Express-session + bcryptjs |
| 👤 จัดการพนักงาน | Role: เจ้าของร้าน / พนักงาน |
| 👥 จัดการสมาชิก | สะสมแต้ม ค้นหาด้วยเบอร์โทร |
| 🍟 จัดการสินค้า | อัปโหลดรูป วิเคราะห์กำไร |
| 🛒 หน้าขาย POS | ตะกร้า โปรโมชั่น ใบเสร็จ |
| 💳 การจ่ายเงิน | เงินสด / โอน / บัตร |
| 📈 รายงาน | 4 ประเภท กรองช่วงเวลา |

## Slide 3: Tech Stack & Architecture
**Title:** เทคโนโลยีที่ใช้พัฒนาระบบ

**Content:**
โปรเจคนี้ใช้ stack ที่เรียบง่ายแต่ทรงพลัง โดยแบ่งเป็น 3 ชั้น:

**Backend:**
- **Node.js + Express.js** — Web server และ routing
- **Mongoose (MongoDB)** — จัดการฐานข้อมูล NoSQL
- **express-session** — จัดการ session สำหรับ login/logout
- **bcryptjs** — เข้ารหัสรหัสผ่านด้วย hashing
- **Multer** — รับอัปโหลดรูปภาพสินค้าและโปรไฟล์

**Frontend:**
- **EJS (Embedded JavaScript)** — Template engine สำหรับ render HTML
- **CSS3 Custom** — ธีมสีอุ่น Warm Burger Theme
- **Chart.js** — กราฟแสดงสถิติใน Dashboard และรายงาน

**Database Models:** User · Member · Product · Sale · Promotion

## Slide 4: ระบบ Login & Role-Based Access
**Title:** ความปลอดภัยด้วย Session และ Role Control

**Content:**
ระบบมีการตรวจสอบสิทธิ์ผู้ใช้ 2 ระดับ:

**เจ้าของร้าน (Owner):**
- เข้าถึงได้ทุกหน้า รวมถึงจัดการพนักงาน โปรโมชั่น และรายงาน
- สามารถยกเลิกบิลและลบข้อมูลได้

**พนักงาน (Employee):**
- เข้าถึงหน้าขาย POS ประวัติการขาย และจัดการสมาชิก
- ไม่สามารถเข้าถึงรายงาน การจัดการพนักงาน และโปรโมชั่น

รหัสผ่านทุกบัญชีถูกเข้ารหัสด้วย **bcrypt hashing** ก่อนบันทึกลงฐานข้อมูล ทำให้ปลอดภัยจากการรั่วไหลของข้อมูล

## Slide 5: หน้าขาย POS (Point of Sale)
**Title:** หน้าขายที่ใช้งานง่าย เลือกสินค้าได้ทันที

**Content:**
หน้าขาย POS ออกแบบให้ใช้งานได้รวดเร็ว โดยแบ่งหน้าจอเป็น 2 ส่วน:

**ส่วนซ้าย — เลือกสินค้า:**
- แสดงสินค้าทั้งหมดในรูปแบบ Grid พร้อมรูปภาพ
- กรองตามหมวดหมู่: เบอร์เกอร์ / เครื่องดื่ม / เมนูเสริม / ของหวาน / คอมโบ
- คลิกเพื่อเพิ่มสินค้าลงตะกร้าได้ทันที

**ส่วนขวา — ตะกร้าและชำระเงิน:**
- ปรับจำนวนสินค้า เพิ่ม/ลด ได้ง่าย
- ค้นหาสมาชิกด้วยเบอร์โทร เพื่อสะสมแต้ม
- เลือกโปรโมชั่น หรือใส่ส่วนลดเองได้
- เลือกช่องทางชำระ: เงินสด / โอน / บัตร
- คำนวณเงินทอนอัตโนมัติ

## Slide 6: จัดการสินค้าและวิเคราะห์ข้อมูล
**Title:** วิเคราะห์กำไรสินค้าแบบ Real-time

**Content:**
โมดูลสินค้าไม่ได้แค่เก็บข้อมูล แต่ยังวิเคราะห์ความคุ้มค่าของสินค้าแต่ละรายการ:

**ข้อมูลที่เก็บต่อสินค้า:**
- ชื่อ, หมวดหมู่, รายละเอียด
- ราคาขาย, ต้นทุน, จำนวนสต็อก
- รูปภาพสินค้า (อัปโหลดด้วย Multer)

**การวิเคราะห์ที่แสดงผล:**
- กำไรต่อชิ้น = ราคาขาย − ต้นทุน
- สถิติรวม: จำนวนสินค้า, สต็อกรวม, ราคาเฉลี่ย
- แจ้งเตือนสินค้าใกล้หมด (สต็อก ≤ 5 ชิ้น)
- ระบบตัดสต็อกอัตโนมัติเมื่อมีการขาย

---

## Slide 7: จัดการสมาชิกและโปรโมชั่น
**Title:** สร้างความภักดีด้วยระบบสมาชิกและโปรโมชั่น

**Content:**
**ระบบสมาชิก:**
- รหัสสมาชิกอัตโนมัติ (MEM0001, MEM0002...)
- สะสมแต้ม 1 แต้ม ทุก ฿10 ที่ซื้อ
- ติดตามยอดซื้อสะสมรายบุคคล
- ค้นหาสมาชิกด้วยเบอร์โทรได้ทันทีในหน้า POS

**ระบบโปรโมชั่น (เจ้าของร้านเท่านั้น):**
- ส่วนลดแบบ % (เช่น ลด 10%)
- ส่วนลดแบบจำนวนเงิน (เช่น ลด ฿20)
- กำหนดยอดซื้อขั้นต่ำ
- กำหนดวันเริ่ม-สิ้นสุดโปรโมชั่น
- ระบบตรวจสอบอายุโปรโมชั่นอัตโนมัติ

## Slide 8: ระบบรายงาน 4 ประเภท
**Title:** รายงานครบ 4 ด้าน กรองช่วงเวลาได้อิสระ

**Content:**
ระบบรายงานออกแบบให้เจ้าของร้านตัดสินใจได้อย่างมีข้อมูล:

| รายงาน | ข้อมูลที่แสดง |
|---|---|
| 💰 รายงานการขาย | ยอดรวม, เฉลี่ยต่อบิล, กราฟรายวัน, สัดส่วนช่องทางชำระ |
| 🍟 รายงานสินค้า | Top 10 ขายดี, สินค้าตามหมวด, สต็อกคงเหลือ |
| 👥 รายงานสมาชิก | Top VIP, แต้มสะสมรวม, ยอดซื้อสะสม |
| 👤 รายงานพนักงาน | ยอดขายแต่ละคน, จำนวนออเดอร์, กราฟเปรียบเทียบ |

ทุกรายงานรองรับการกรองตามช่วงวันที่ (Date Range Filter) และแสดงผลด้วยกราฟ Chart.js

## Slide 10: สรุปและการใช้งาน
**Title:** ระบบพร้อมใช้งาน ติดตั้งง่าย ใน 3 ขั้นตอน

**Content:**
**การติดตั้ง:**
1. `npm install` — ติดตั้ง dependencies
2. `node seed.js` — สร้างข้อมูลเริ่มต้น
3. `npm start` — เปิดเซิร์ฟเวอร์ที่ port 3000

**บัญชีเริ่มต้น:**
- เจ้าของร้าน: `admin` / `admin123`
- พนักงาน: `emp01` / `emp123`

**สิ่งที่ระบบทำได้:**
- ✅ จัดการพนักงาน 2 Role (เจ้าของ/พนักงาน)
- ✅ จัดการสมาชิก + สะสมแต้ม
- ✅ จัดการสินค้า + อัปโหลดรูป + วิเคราะห์กำไร
- ✅ ขายหน้าร้าน POS + โปรโมชั่น + ใบเสร็จ
- ✅ รับชำระ 3 ช่องทาง + คำนวณเงินทอน
- ✅ รายงาน 4 ประเภท + กรองช่วงเวลา
